# DailyJournal

## Running the App

1. Clone this repository to your local machine:

    ```
    git clone [DailyJournal]
    ```

2. Navigate to the project directory:

    ```
    cd [DailyJournal]
    ```

3. Get the dependencies:

    ```
    flutter pub get
    ```

4. Connect your device or start an emulator.

5. Run the app:

    ```
    flutter run
    ```
